# Trinko
A simple application made for the Google Associate Developer Certification.
It consists of three activites: 
 - First activity: list of favourites cocktails. You can remove them by swiping on the cardview
 - Second Activity (Opened by pressing the FAB): Search the cocktail from a REST service by www.thecocktaildb.com
 - Third Activity (Opened by selecting a cocktail on the second activity): See the cocktail details like ingredients and the recipe.
